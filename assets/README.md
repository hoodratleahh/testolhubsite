# Assets

## iPhone mockup

The site uses a free iPhone mockup from [Unsplash](https://unsplash.com) (hotlinked). To use your own image:

1. Add `iphone-mockup.png` or `iphone-mockup.jpg` to this folder
2. In `index.html`, change the Discord section `img` src to `assets/iphone-mockup.png` (or .jpg)

`iphone-mockup.svg` is a minimal CSS fallback frame you can customize.
